### Run Locally

To run **Homeverse** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/satyendra-coder/BigYard.git
```

Windows:

```bash
git clone https://github.com/satyendra-coder/BigYard.git
```
